# LMAH Fine Foods

Wholesale trade website for LMAH — European Fine Foods, Cape Town.

## Contents

- `index.html` — Complete single-file website with all images embedded as base64

## Deployment

This is a single self-contained HTML file. It can be deployed to:

- **GitHub Pages** — push to a repo, enable Pages, done
- **Netlify** — drag and drop the folder into netlify.com/drop
- **Vercel** — `vercel deploy` from this directory
- **Any static host** — upload index.html

## Trade Password

The catalogue price unlock password is: **LMAH2025**
Change it by searching for `const PW = 'LMAH2025'` in index.html.

## Notes

- All product images are embedded — no external dependencies
- No build step required
- Works offline once loaded
